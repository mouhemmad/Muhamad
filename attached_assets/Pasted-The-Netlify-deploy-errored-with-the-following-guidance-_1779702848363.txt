Diagnosis

- Relevant log lines: [line 54](#L54) shows npm reporting "Missing script: "build⁠"", and [line 72](#L72) shows Netlify's configured build command is `npm run build⁠` (note the invisible/extra character after build). The failure originates from npm not finding a "build" script in package.json (or the command being different due to a hidden character), causing the build to exit with code 2.
- Error type and cause: Missing npm script / bad build command. Either:
  1) package.json does not include a "build" script, or
  2) the Netlify build command includes a hidden/Unicode character so npm is looking for a script literally named "build⁠" (with an extra character), which does not exist.

Verify package.json
- Open your repo package.json to confirm available scripts: https://github.com/mouhemmad/Muhamad/blob/main/package.json

Solution

1) Quick check for the hidden character (most likely):
   - In the Netlify App > Site settings > Build & deploy > Continuous Deployment > Build settings, edit the Build command. Delete the current value and retype exactly:
     npm run build
   - Save and redeploy. This removes any stray invisible Unicode character that makes npm search for a different script name (this is the simplest fix if your package.json actually has a "build" script).

2) If your project has no build step (static site):
   - Remove the build command in Netlify (leave it blank) or set it to an empty string, and set the Publish directory to the folder containing your site (e.g., "/" or "public" if your files live there). Then redeploy.

3) If your project requires a build step but package.json currently lacks a "build" script:
   - First verify the package.json in the repo contains the correct scripts (see link above).
   - Add an appropriate build script to package.json and commit. Examples (choose the one matching your toolchain):
```json
{
  "scripts": {
    "build": "vite build"         // for Vite projects
  }
}
```
```json
{
  "scripts": {
    "build": "react-scripts build" // for Create React App
  }
}
```
```json
{
  "scripts": {
    "build": "next build"         // for Next.js (and set up proper adapter)
  }
}
```
   - Commit and push package.json, then redeploy on Netlify.

Notes
- If you change the build command in Netlify, be careful to re-type it (do not copy/paste) to avoid invisible characters.
- If you do need to change Node versions, see Netlify docs for Node version management: https://docs.netlify.com/configure-builds/manage-dependencies/#node-js-and-javascript
